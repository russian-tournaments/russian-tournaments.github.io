Dota 2 турнир Game Show Global eSports Cup Season 1

Турнир проходил с 2015-09-14 по 2015-12-06
Призовой фонд 330,322

Game Show Global Esports Cup будет состоять из трех сезонов, которые пройдут в 2015-2016-х годах, каждый из которых будет иметь стартовый призовой фонд в $330,322 + полный трэвел саппорт для команд на оба ЛАН финала (студийный и гранд-финал)

Описание турнира

Формат турнира состоит из шести основных частей. Открытые онлайн квалификации, далее последуют 2 групповых этапа, из которых лучшие команды своего региона попадут в стадию плей-офф. Отборочные игры будут вестись в 5 регионах: Северная Америка, Европа, СНГ, Китай и ЮВА.

Открытая квалификация

  • Записаться на открытую квалификацию можно здесь
  • Первый этап, состоящий из четырех открытых квалификаций, пройдет с 14.09.2015 по 21.09.2015, на каждую из квалификаций отводится два дня. Формат матчей «Best of 1» до стадии четвертьфинала (топ 8 команд), игры за квоту будут проходить в формате «Best of 3».

CIS - Стадия 1

  • 16 команд разделены на 4 группы
  • Каждая группа - double elimination сетка
  • Топ 2 команды из каждой группы проходят в стадию 2

CIS - Стадия 2

  • 8 команд из первой стадии играют друг с другом с 23 по 27 сентября 2015
  • Все матчи best-of-three
  • Top 2 команды проходят в стадию 3

CIS - Стадия 3

  • 2 команды из стадии 2 присоединяются к приглашенным командам
  • Все матчи best-of-two
  • Игры будут проходить с 1 по 4 октября 2015 на выставке «Игромир»
  • Топ 4 команды проходят в плей-офф

Лучшие 8 команд сойдутся в рамках Studio ивента в Вильнюсе (Литва), на площадках новых профессиональных студий компании Game Show 7-10 декабря.

LAN-финалы с лучшими 4 командами по итогам Studio ивента пройдут 5-6 декабря в Москве.
